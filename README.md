# LiminalDeep

A pool-based liminal horror Bukkit mod for Minecraft. Find out what lurks beneath...

## Installation Instructions

Download the latest release, and put the .jar file in your server's plugins folder.

Now edit the bukkit.yml file in your server folder, and add the following section at the bottom:

```
worlds:
  world_pools:
    generator: LiminalDeep
```

This will tell the server to use the LiminalDeep plugin to generate the world "world_pools".
Change that name if you want to use a different world.
